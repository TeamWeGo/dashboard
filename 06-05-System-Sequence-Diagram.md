# System Sequence Diagram （功能模型）

根据我们的[用例分析](https://teamwego.github.io/dashboard/06-02-use-cases.html)

## 基本功能

1. 发布者发布任务
2. 接受者接受任务

发布者可以在任务发布平台上填写任务信息发布自己的任务，接受者在浏览到任务清单之后选择性的接受自己的任务。
