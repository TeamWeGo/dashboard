# UI Design

## UI 结构
###　 首先设计整个小程序一共有7个界面
![](https://raw.githubusercontent.com/zxftuid/picbed/image/img/%E6%8C%A3%E9%97%B2%E9%92%B1.png)

### 主要的页面有4个


* 首页　展示任务推荐，任务列表，任务搜索，任务种类
* 任务　展示用户发布，确认，结束，加入，正在做，结束的任务
* 发布　用户在本界面发布任务以及发布调查问卷
* 个人　展示个人信息，编辑个人喜好和签名，查看闲钱币
